# **BORNTOWIN**

## 🍀 프로젝트 주제 🍀
짐웨어 쇼핑몰 
<br>

## 🔅 프로젝트 진행 기간
**2024.04.23 ~ 2024.05.07** 프로젝트 일정

## 🌈 개발환경
- Eclipse
- Apache Tomcat
- Oracle SQL Developer

## 🔅 사용 기술
- JAVA, JSP, Oracle, HTML5, JavaScript, CSS3, jQuery

## 👉 주요 기능
- 로그인
- 회원가입
- 메인페이지
- 제품 목록
- 제품 상세 페이지
- 장바구니
- 결제
- 마이페이지
- 게시판(공지, Q&A)
- 관리자 페이지(상품 관리, Q&A 관리)
